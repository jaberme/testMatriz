# Demostración de llamadas a la Interfaz Nativa

Este proyecto se desarrolla para la asignatura **Programación Nativa en Dispositivos Móviles**.

Cuenta con **cinco versiones de código**:

1. **Java (single thread):** servirá para demostrar *“el peso”* de la máquina virtual, y veremos cómo **JIT** acelera la ejecución.  
2. **Java (threaded):** servirá para analizar la gestión de las **green threads** por parte de la JVM.  
3. **C:** código C, **una hebra**.  
4. **C:** código C, con **varias hebras (pthreads)**.  
5. **Uso de Bionic (memcpy).**
